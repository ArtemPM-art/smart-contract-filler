# Smart Fill v3.3

Автоматизация заполнения спецификаций и коммерческих предложений.
Сокращает время подготовки документа с нескольких часов до 30 минут, исключая ошибки ручного счета.

## Функции

- **Полный пересчет таблиц:** Автоматически находит таблицы и рассчитывает `Количество × Цена(Сумма с НДС 22%) = Сумма(Цена без НДС)`. Подсчитывает итоги по каждой таблице.
- **Генерация суммы прописью:** Автоматически добавляет итоговые строки под таблицами и в конце раздела с расшифровкой суммы текстом (например, *"Один миллион... 00 копеек"*).
- **Два режима работы:**
  - **Стандарт (STD):** В общую стоимость включается Оборудование, ПНР и ПО.
  - **Проект ИМ (IM):** ПО исключается из общей стоимости, и итоговая сумма размещается *перед* блоком с лицензиями.
- **Стандартизация оформления:** Все вставляемые цифры приводятся к единому шрифту **Times New Roman**, что обеспечивает аккуратный вид документа.
- **Подробное логирование:** Каждое действие программы (что увидела, что посчитала, что записала) сохраняется в текстовый лог-файл для полного контроля над процессом.
- **Защита от ошибок:** При сбоях или отсутствии необходимых компонентов программа показывает понятное окно с ошибкой, а не "падает" молча.

## Запуск

### Для пользователя (запуск .exe)
1. Откройте файл `SmartFill_Start.exe`.
2. Выберите необходимые настройки в появившемся окне.
3. Нажмите кнопку **"ПРИМЕНИТЬ И ВЫБРАТЬ ФАЙЛ"**.
4. Укажите `.docx` файл для обработки.
5. Готовый файл с припиской `_ГОТОВЫЙ_` появится в той же папке.

### Для разработчика (запуск из кода)
